# Odoo - Añadir campo al formulario de contacto

Módulo personalizado para Odoo que añade un campo extra al formulario de contacto del sitio web.

## Características

- Añade un campo personalizado al formulario de contacto.
- Los datos se guardan en la base de datos.
- Compatible con Odoo (especifica tu versión, ej: 16.0, 17.0).

## Instalación

1. Copia el módulo en la carpeta `addons` de Odoo.
2. Actualiza la lista de módulos.
3. Instala el módulo desde *Apps*.

## Uso

Una vez instalado, el campo aparecerá automáticamente en el formulario de contacto del sitio web.
